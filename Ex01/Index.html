HTML basics

HTML builds the structure of a page. Tags like <h1>, <p>, <a>, and <img> are common.

<!-- simple structure -->
<h1>Hello world</h1>
<p>This is a paragraph. <a href="https://example.com">Link</a></p>
<img src="path/to/image.jpg" alt="description">

CSS basics

CSS styles HTML. Rules target elements and change colors, layout, and spacing.

/* select elements */
h1 { color:#1f2937; font-size:28px; }
.container { max-width:800px; margin:0 auto; padding:16px; }

JavaScript basics

JavaScript makes pages interactive. Use console.log to debug in the browser console.

// simple example
const btn = document.querySelector('button');
btn?.addEventListener('click', () => {
    console.log('Clicked!');
});
